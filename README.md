# 🎓 Student Grade Management System (SGMS)

A simple Java console application developed to practice core programming fundamentals.

---

## 📌 Project Overview

This project allows users to input student information and subject grades, then calculates:

- Total Marks
- Average Marks
- Percentage
- Pass/Fail Status
- Final Grade (A, B, C, D, F)

It is built as a practice project to strengthen Java basics and logical thinking.

---

## 🧠 Features

- Take student name and age
- Input marks for:
  - Mathematics
  - English
  - Science
- Calculate:
  - Total marks
  - Average
  - Percentage
- Determine:
  - Pass or Fail status
  - Final grade based on average

---

## 🛠️ Technologies Used

- Java
- Scanner Class (User Input)
- Conditional Statements (if / else if / else)
- Data Types (int, double, boolean, char)

---

## 📊 Grading System

| Average Range | Grade |
|--------------|------|
| 90 - 100     | A    |
| 80 - 89      | B    |
| 70 - 79      | C    |
| 60 - 69      | D    |
| Below 60     | F    |

## 🚀 How to Run

1. Clone the repository
2. Open in any Java IDE (IntelliJ / Eclipse / VS Code)
3. Run `Main.java`
4. Enter student details when prompted
